🥗 ¿Qué es una dieta?

Una dieta no es "comer poquito" ni privarse de lo que te gusta. Es la forma en la que te alimentas cada día, con el objetivo de darle a tu cuerpo la energía y nutrientes que necesita para funcionar y rendir al máximo.

No solo importa cuánto comes, sino también qué comes y cómo lo distribuyes a lo largo del día. Por eso se calculan tus calorías y macronutrientes (proteínas, grasas y carbohidratos).

📌 ¿Qué significa cada cálculo?
{% if dieta.calculos.imc %}

IMC (Índice de Masa Corporal): Relación entre tu peso y tu altura. Rangos: bajo (<18.5), normal (18.5–24.9), sobrepeso (25–29.9), obesidad (>30). No diferencia músculo y grasa.

{% endif %} {% if dieta.calculos.porcentaje_grasa %}

% Grasa Corporal: Qué parte de tu peso corresponde a grasa. Clave para definir si el enfoque es quemar grasa, ganar músculo o mantener.

{% endif %} {% if dieta.calculos.tmb %}

TMB (Tasa Metabólica Basal): Calorías mínimas que tu cuerpo necesita en reposo total (respirar, corazón, temperatura). Es la base del cálculo.

{% endif %} {% if dieta.calculos.tdee %}

TDEE (Gasto Calórico Diario Total): Gasto real considerando tu actividad física. Desde aquí se define tu déficit, mantenimiento o superávit calórico.

{% endif %}
